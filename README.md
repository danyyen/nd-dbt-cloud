# nd-dbt-cloud

## Aim:
This project series provides a comprehensive understanding of the Data Build Tool (DBT). The initial project focuses on DBT fundamentals, including its purpose, place in the ETL process, data modeling, and logging. Using Netflix movie and series data in Snowflake, the project will demonstrate DBT's application in real-world scenarios and highlight its features for building robust data pipelines. This will help users optimize their data transformation and modeling workflows.

## Tech Stack:
*   **Language:** Python, SQL
*   **Tool:** DBT
*   **Database:** Snowflake
*   **Services:** AWS EC2

## Key Takeaways
*   Understanding DBT fundamentals, its necessity, and details.
*   DBT installation on Windows and EC2.
*   Creating DBT projects and understanding their structure.
*   Uploading raw data to Snowflake using Python.
*   Exploring data modeling, profiles, and logging in DBT.
*   Using Materializations and Variables.
*   Implementing Sources, Prehooks, and Posthooks.
*   Developing models to address business questions and executing them.
*   Understanding different types of testing in DBT.
*   Comparing DBT Cloud and DBT Core.
